## 🚀 Installation

1. Clone the repository:

```sh
git clone https://github.com/yusufardapeker/product-list.git
cd product-list
```

2. Install dependencies:

```sh
npm install
```

3. Start the development server:

```sh
npm run dev
```

4. Open your browser and go to http://localhost:5173
